Child Locator App
General Terms FAQ Download
Child Locator is an application that allows a parent to know the current location of another person, usually a child, for parenting or similar purposes. Locations are shared and displayed on request so that battery and device resources are only used when needed.

The user is the only responsible person for any misuse of the app.
