Privacy policy

What information does Child Locator collect? Child Locator app does not collect any kind of information. Account information provided by the user are stored in a server for validation purposes. Locations, statistics or usage data of the app are not collected.

What information does Child Locator use? Child Locator app uses registered information for validation purposes and communication with the user when needed. Location information is only displayed on the corresponding parent device but never stored by the app.

What information does Child Locator share? Location information is shared from child devices to parent devices upon request through a secured communication channel by using TLS protocol.

Consent to the processing of personal data

Each user of the app has to provide the following data when registering an account: Name, email, username and password. The purpose of each data is as follows:
  • Email: Can be used as a part of login credentials as well as a way to communicate with the user for the purpose of updates or to recover/change a password.
  • Username: Can be used as a part of login credentials.
  • Name: Used to address the user in different kinds of communication with the platform and when displaying information to on the app.
  • Password: Used as a part of login credentials.
All new data generated by the user on the app such as adding new children or subscription data are recorded in our server and identified with the data provided in the registration form. Owners of the server as well as app administrators have access to the information provided by each user on the app. All provided data is stored in a database on a server. The data is accessed each time it is needed such as, but not limited to, the following cases:
  • when a user opens the app,
  • when a user adds a child
  • when a user subscribes
  • when administrators are managing the app or the server
  • when there is a need to communicate with a user
The app does not store payment credentials when processing payments for subscriptions. Payment data is processed by Google Play. Nevertheless, the app has access to partial information of transactions concerning a subscription displayed by Google Play. Location data is accessed on a device that is currently login as a child and has granted all the permissions to the app. Location data is only accessed each time a parent device requests its location. Location data is only displayed but not stored in the app.
